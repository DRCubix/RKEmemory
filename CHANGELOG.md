# Changelog

## [1.0.0] - 2026-04-24

### Initial Release

- **RLM Router** — Recursive Language Model routing with peek/grep/partition operations
- **LLM-Wiki** — Git-tracked markdown knowledge base with ingest/query/lint
- **Vector Store** — Qdrant + BGE-M3 semantic search (1024-dim embeddings)
- **Knowledge Graph** — FalkorDB with Cypher queries and GraphRAG
- **Agent Integration** — Context injection for Claude, Codex, Gemini, and custom agents
- **Ingestion Pipeline** — Parallel knowledge ingestion from files, Drive, and git repos
- **CLI** — Full command-line interface for all operations
- **Docker Compose** — One-command service deployment
- **Bare-metal support** — Native deployment without Docker

RKE — Query Flow: "How do we handle OAuth?"

Step-by-step data flow from agent question to enriched answer

① Wiki Search — Keyword

  • • Scans the wiki index for exact term matches
  • • Fast sub-millisecond response
  • • Returns full page content for matched entries
  • • Cross-references related pages via tags

② Vector Search — Semantic

  • • BGE-M3 embeds the query into 1024-dim vector
  • • Qdrant finds most similar content chunks
  • • Hybrid: keyword boosting for page titles
  • • Returns top-N results with similarity scores

③ Graph Search — Entities

  • • Cypher queries extract entity relationships
  • • Finds connected APIs, accounts, decisions
  • • GraphRAG: extract subgraph for LLM synthesis
  • • Returns structured entity data with links

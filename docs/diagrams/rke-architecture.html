RKE — Recursive Knowledge Engine

Unified multi-agent memory with RLM routing, LLM-Wiki, Qdrant vectors & FalkorDB graph

RLM Router — No Context Rot

  • • Agents explore knowledge via peek(), grep(), partition()
  • • Recursive depth control with cost limits
  • • Based on Zhang & Khattab (2025) paper
  • • Agents never see full context — always focused

Multi-Layer Search

  • • Keyword: fast exact matching on wiki index
  • • Semantic: BGE-M3 vector similarity (1024d)
  • • Graph: Cypher entity relationships via FalkorDB
  • • Results synthesized into unified context block

Cross-Agent Shared Memory

  • • Claude, Codex, Gemini, Hermes all share one KB
  • • Knowledge compounds — every query enriches the system
  • • Git-tracked wiki with full version history
  • • Local-first: no API calls, no data leaves your machine
